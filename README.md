# physics-analyzer
物理動画解析ミニアプリ

index.html をChromeで開くと動きます。動画データは端末内で処理され、外部サーバーには送信されません。

主な機能
- 動画読み込み
- 再生・停止
- シークバーでの移動
- 少し進む／少し戻る
- 計測スタート／計測終わりの設定
- 原点設定
- スケール設定
- 物体位置のクリック記録
- 記録点の軌跡表示と番号表示
- 記録後の自動フレーム送り（1,2,5フレーム相当）
- t, x, y の表表示
- CSV保存／CSVコピー

モード
- 自由落下モード：主に t-y の関係を調べる授業向け。表には t, x, y を残します。
- 斜方投射モード：t-x と t-y を分けて考える授業向け。軌跡表示で放物線状か確認できます。

使い方
1. 動画を選択する。
2. シークバーで開始場面を探し、「ここから計測スタート」を押す。
3. 終了場面で「ここで計測終わり」を押す。
4. 基準長さ[m]を入力し、「スケールを設定」で動画内の基準長さの両端をクリックする。
5. 「原点を設定」で原点にしたい場所をクリックする。
6. 「物体の位置を記録」を押し、物体の中心をクリックしていく。
7. CSV保存またはCSVコピーでデータを取り出す。
